Key Flows

Sale flow: Tap tile (+1 badge) → tap Charge → PaymentModal → tap denominations → change shown → Confirm → saved to SQLite → cart clears
Void during order: Long-press tile → removes item (no password needed, order not yet paid)
Void completed sale: TransactionsModal → tap transaction → Void → AdminModal password prompt → confirmed → status = 'voided'
Product creation: ProductsModal → + New → name, price, emoji → Save → appears in tile grid
